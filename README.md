A modern, fully responsive grocery shopping website interface with product cards, category filters, search functionality, and clean gray-themed design.
## Features

- Fully Responsive - 2 products per row on mobile, 4 on desktop
- Search Bar - Integrated in navigation header
- Category Tabs - Scrollable categories (All, Produce, Dairy, Meat, etc.)
- Product Cards - Each product shows category, name, shop, price, description
- Hero Banner - Eye-catching grocery promotion section
- Call-to-Action - Start Shopping button with fresh groceries card
- Footer - Shop links, support info, app download icons
- Indian Rupee (Rs) - Prices displayed in INR
- Modern Gray Theme - Professional neutral color palette
- No JavaScript Required - Pure HTML/CSS with Bootstrap

## Responsive Breakpoints

| Device | Screen Width | Products Per Row |
|--------|--------------|------------------|
| Mobile | 576px and below | 2 |
| Tablet | 768px to 991px | 3 |
| Desktop | 992px and above | 4 |
