✨ Features
🛒 Shopping Cart
Add to Cart - Click the "Add to Cart" button on any product

Quantity Management - Increase or decrease item quantity directly in the cart

Remove Items - Delete individual items from the cart

Real-time Updates - Cart count badge updates instantly

Total Calculation - Automatically calculates total items and total price

Checkout - Complete your order with a single click

Toast Notifications - Visual feedback for all cart actions

🔍 Product Features
Category Filtering - Filter products by category (Dairy, Produce, Meat, Bakery, etc.)

Search Functionality - Search products by name or category

Product Details - View product information with a single click

Responsive Grid - 2 columns on mobile, 3 on tablet, 4 on desktop

Product Images - High-quality product images (fallback included)

📄 Pages
Home Page - Product grid with all features

Contact Page - Working contact form with validation

FAQ Page - Accordion-style FAQs with search functionality

🎨 UI/UX
Sticky Header - Always accessible navigation

Sidebar Cart - Slide-out cart for better UX

Toast Messages - Non-intrusive notifications

Mobile-First Design - Optimized for all screen sizes

Clean Typography - Inter font family for modern look
